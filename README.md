## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nexus1343/it-transfer.git
   cd it-transfer
   ```

2. **Install dependencies**

