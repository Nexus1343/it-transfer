## Technical Stack
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS + shadcn/ui components
- **State Management**: React Context/Zustand
- **Icons**: Lucide React
- **Charts**: Recharts (for analytics)
- **Mock Data**: JSON files with TypeScript interfaces
